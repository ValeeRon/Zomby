Чтоб развернуть этот проект, необходимо сделать следующие действия:
1.Скачать данный проект из репозитория в виде архива или клонировать с помощью git
2.Выполнить команду php artisan key:generate для папки с проектом
3.Провести миграции с помощью команды php artisan migrate
4.Заполнить базу данных с помощью команды php artisan db:seed
5.Создать файл с названием ".env" в корне проекта
6.Скопировать содержимое файла .env.example в файл .env
7.Заменить значение параметра DB_DATABASE на zomby
8.Заменить параметры DB_USERNAME и DB_PASSWORD на свои
9.Выполнить команду php artisan serve для папки с проектом
10.Перейти по указаному адресу
Готово!
